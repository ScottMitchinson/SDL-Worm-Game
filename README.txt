Goal:
Try to stop the worms from reaching the apple by launching objects at them.

Controls:
Aim with the mouse (if the line is longer than around 10cm, it can cause problems with the aim, so try to keep it close)
	The closer the mouse to the line start the less power it has.

Left click: Fire.

Right click: Wind Ability (Launches all enemies on the screen with a force).

D: debug mode, will show the path planning line and the collision boxes.

A: When Debug mode is active, will spawn an enemy at the mouse position.
